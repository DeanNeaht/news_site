Документация API

Введение

Public API и Private API представляют собой удобное средство взаимодействия с сайтом стороннего програмного обеспечения.

-Public API используется для получения информации, не требущая доступа к личной информации аккаунта

-Private API используется для добавления, изменения, удаления новостей, а так же для получения информации, для которой необходим доступ к личным данным аккаунта.

Public API

В настоящий момент Public API существует в одной единственной версии.

Возможности:

  • Получение всех новостей
  • Получение новостей в определенной категории
  • Получение конкретной новости
  • Получение новостей опреденного пользователя

Получение всех новостей

Чтобы получить все новости, необходимо перейти по ссылке:

https://dogy-bot.xyz/news/

Получение новостей в определенной категории

Чтобы получить все новости в определенной категории необходимо, перейти по ссылке:

https://dogy-bot.xyz/news/{category}/

Список категорий:

  • politics
  • culture
  • science
  • society
  • economy
  • other

Получение конкретной новости

Чтобы получить конкретную новость, необходимо знать id новости:

https://dogy-bot.xyz/news/detail/{id}/

Получение новостей определнного пользователя

Все новости пользователя можно узнать по его username.

https://dogy-bot.xyz/user/{username}/

Private Api

Private API необходим для осуществления создания, редактирования, удаления новостей, а так же получения информации, для которой необходим доступ к личным данным аккаунта.

Для использования Trade API необходимо получить токен в личном кабинете.
Токен используется для аутентификации пользоваткля.

Адрес для отправки запросов Trade API: 

Для каждого запроса нужна аутентификация.

Все ответы сервер отдаёт в формате JSON
